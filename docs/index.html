Documentation du Projet IA

Description

Ce document décrit le projet IA incluant les étapes d'entraînement, validation, test et déploiement du modèle.

Structure du Dépôt

  • script/ : Scripts Python (entraînement, validation, déploiement, envoi d'email, etc.).
  • notebook/ : Jupyter Notebooks pour analyses exploratoires.
  • model/ : Modèles entraînés et déployés.
  • data/ : Données utilisées dans le projet (suivies avec Git LFS).
  • doc/ : Documentation générée automatiquement.
  • keys/ : Fichiers de configuration et secrets.

Projet réalisé par l'équipe IA de NTONGA, ISSA, SIMO.
